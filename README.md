# Sudoku

A modernisation of a classic Win32 Sudoku program.

The aim of this project is to preserve the original solver while
carefully modernising the user interface and codebase.

## Current status

- x64 build
- Modern Win32 callback signatures
- Legacy Win16 declarations removed
- Visual Studio 2017 project

## Planned improvements

- Simplified UI architecture
- Renderer-based Sudoku board
- 9×9 and 16×16 support
- Cleaner helper functions
