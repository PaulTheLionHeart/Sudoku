# Sudoku Quick Reference

This page provides a quick summary of the most commonly used commands and mouse actions.

For complete explanations, examples and tutorials, see [Sudoku Help](UserGuide.md).

---

# Keyboard Shortcuts

| Key | Action |
|-----|--------|
| **S** | Select puzzle type (9×9 or 16×16) |
| **C** | Create a new blank puzzle |
| **G** | Generate a new Sudoku puzzle |
| **Esc** | Close the current dialog |

---

# Mouse Actions

## 9×9 Sudoku

| Action | Result |
|--------|--------|
| Left-click candidate | Enter that value |
| Left-click completed cell | Undo the value |
| Right-click completed cell | Toggle Original/User |

## 16×16 Sudoku

| Action | Result |
|--------|--------|
| Left-click empty cell | Open hexadecimal value picker |
| Left-click completed cell | Undo the value |
| Right-click completed cell | Toggle Original/User |

---

# Main Window Buttons

| Button | Purpose |
|--------|---------|
| **Select** | Choose 9×9 or 16×16 Sudoku |
| **Create** | Create a new blank puzzle |
| **Generate** | Generate a new Sudoku puzzle |
| **Original Puzzle** | Toggle original clue entry mode |
| **Solve** | Solve the current puzzle |
| **Refresh** | Clear the board and begin again |
| **Save Sudoku** | Save the puzzle in native Sudoku format |
| **Save CSV** | Export the puzzle as a CSV file |
| **Finish** | Close the Sudoku editor |

If an **Impasse** message appears, the current puzzle contains an inconsistency and no valid solution can be found.

---

# Puzzle Generation

Press **G** or choose **Generate Sudoku**.

Select:

- Puzzle size
- Difficulty
- Symmetry

Press **Generate** to create a new puzzle.
